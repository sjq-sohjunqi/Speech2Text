What is Speech2Text?

Speech2Text is a transcription service that allows users to transcribe audio recordings of their meetings into text format. The system is designed to make the process of logging project meetings a simple task.

Speech2Text also allows users to manage and share these transcripts with other members of their team.

Document your project meetings today with Speech2Text!

Sharing Transcripts

Note: This feature is only available to users who have an account

Transcripts can be shared with users and groups. The following permissions can be given when sharing:
  • Read-Only: Only able to view the transcript and its annotation
  • Read & Write: Able to edit and save the transcript and its annotation

*Do take note that if a transcript is shared with a user and a group that user is in, the permission that takes precedence is the one with the most privilege.

Privilege from highest to lowest:
Read & Write > Read-Only > Not Shared

Sharing with users

1 Go to the transcripts page using the icon on the navigation bar above.
